INSTRUCTIONS
The following software is required in order to use the files 
in the Appendix B Chapter 12 Supplement folder: MATLAB Version 7 and MATLAB's Control System Toolbox Version 8. Follow the 
installation instructions included with MATLAB.

Be sure the folder "MATLAB files for Appendix B for Chapter 12" is added to the search path in Set Path... under the File menu in the MATLAB window or the M-files appear in the MATLAB Current Folder window. 

To see the computer responses after installing the M-files, 
run each problem by typing the M-file name, such as ch12p1, 
after the prompt (>>) in the Command window. To view all or 
part of the M-file in the MATLAB Command window, enter 
"type <file name>" or "help <file name>", respectively, after 
the prompt.
